Privacy Policy — SmartSpend Analyzer (College Project)

Last updated: 2025-11-30

1. Introduction

This site and the SmartSpend Analyzer app are used only for a college project. We only collect public Page metadata (page name, category, followers count) and public post engagement (reactions, comments count, shares) from Facebook Pages. No personal user data is collected or stored.

2. Data collected

We may read and temporarily store: Page name, Page category, followers_count, post id, created_time, message text, reactions_count, comments_count, shares_count.

3. Purpose & usage

Data is used solely for academic analysis and demonstration in classroom reports. We will not publish data publicly or collect private Facebook user data.

4. Data retention

Data is stored only for the duration of the analysis and may be exported to a CSV for classroom use. Data will be deleted after project submission.
